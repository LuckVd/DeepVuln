## 裁决规则

| 规则 | 条件 | 结果 |
|------|------|------|
| 1 | NOT_EXPLOITABLE | 无条件覆盖为 not_exploitable |
| 2 | UNLIKELY + HIGH/CRITICAL | 降级为 CONDITIONAL |
| 3 | EXPLOITABLE + HIGH/CRITICAL | 确认为 EXPLOITABLE |
| 4 | LIKELY/POSSIBLE | 默认 CONDITIONAL |
| 5 | 无 exploitability | 默认 CONDITIONAL |

## 冲突禁止机制

- NOT_EXPLOITABLE + EXPLOITABLE 状态并存 → ArchitectureViolationError
- validate_no_conflict() 检测逻辑错误
