## 技术细节

### 评分维度

1. **CodeQL 数据流** (60% 权重)
   - 完整数据流 (source + sink): 0.8-1.0
   - 部分数据流: 0.5-0.7
   - 有消毒剂: 0.0-0.3
   - 无数据流: 0.0-0.2

2. **污点追踪** (20% 权重)
   - is_exploitable: 0.95 × path_factor
   - is_sanitized: 0.0-0.2
   - not_reachable: 0.0

3. **可达性** (15% 权重)
   - HTTP 入口点: 1.0
   - CLI 入口点: 0.8
   - 有调用者: 0.5-0.7
   - 孤立函数: 0.1

4. **攻击面** (5% 权重)
   - HTTP/API: 1.0
   - CLI: 0.7
   - Library: 0.3

### 融合策略

- **WeightedAverage**: 加权平均，自动归一化
- **Conservative**: 取最小值（风险厌恶）
- **Optimistic**: 取最大值（覆盖优先）

### 置信度计算

```
avg_confidence = 可用维度置信度平均值
missing_penalty = 0.95 ^ 缺失维度数
dimension_boost = [0.75 (1维) / 0.9 (2维) / 1.0 (3+维)]
final_confidence = avg_confidence × missing_penalty × dimension_boost
```
