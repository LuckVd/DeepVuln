## 技术细节

### 判定结果类型
- CONFIRMED: 漏洞真实可利用
- FALSE_POSITIVE: 存在有效防御措施
- CONDITIONAL: 特定条件下可利用
- NEEDS_REVIEW: 需人工审查

### Prompt 设计
- 攻击者 Prompt：含绕过技术参考库
- 防御者 Prompt：含防御机制参考库
- 仲裁者 Prompt：含判定标准

### 静态分析
- DefenderVerifier 内置多语言 sanitizer 模式匹配
- 支持 Python/JavaScript/Java/Go 四种语言

## 测试结果
