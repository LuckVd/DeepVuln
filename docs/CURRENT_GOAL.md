## 问题背景

### 问题：环境依赖复杂

**现象**：
- CodeQL 需要 Go/Java/Node.js 等构建环境
- 不同项目需要不同的语言版本和工具
- 用户配置环境成本高

**解决方案**：
1. 创建包含所有依赖的 Docker 镜像
2. 支持多语言项目扫描
3. 一键运行，无需配置
